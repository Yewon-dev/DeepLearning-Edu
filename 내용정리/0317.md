## Data Mining

- 대규모로 저장된 데이터 안에서 체계적이고 자동적으로 통계적 규칙이나 패턴을 찾아 내는 것



#### Data Checkpoints

- Volume
- Timeliness
- Variety
- Quality
- Relevance

-----



### 데이터 분석의 유형

1. One-off 분석
   - 일회성의 비즈니스 질문에 대한 답 제공
   - 사전에 형식과 범위가 정해져있지 않음
   - 해석이 중요
2. Production System
   - 동일한 구조의 분석을 지속적으로 반복



### 데이터 분석 절차

- 분석 주제 결정 -> 후보 가설 도출 -> 관련 데이터 확보 -> 데이터 통합/정리 -> 분석/모델링 -> 결과 도출
