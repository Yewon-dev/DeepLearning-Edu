## Machine Learning

1. 지도학습 Supervised Learning

   - 레이블이 있는 훈련데이터로 모델을 학습하여 미래의 데이터에 관한 예측을 만드는 것

   - 분류모델, 회귀 모델

2. 비지도학습 Unsupervised Learning

   - 레이블이 없는 데이터, 데이터가 어떻게 구성되었는지를 알아내는 문제의 범주
   - 군집 모델, 차원 축소

3. 강화학습 Reinforcement Learning
   - 환경과의 상호작용을 통해 자신의 성능을 개선하는 시스템
